# ✨ Features

### 🔍 Smart News Search

Search the latest headlines using any keyword or topic.

Examples:

- Artificial Intelligence
- Cyber Security
- Football
- NASA
- Bitcoin

---

### ⚙️ Advanced Filters

Customize your search using:

- Sort by Popularity, Relevancy or Published Date
- Language Selection
- Adjustable Number of Articles
- Time Range Filter

---

### 📰 Beautiful News Cards

Every article includes:

- 🖼 Featured Image
- 📰 Title
- 🏢 Source
- ✍ Author
- 📅 Publication Date
- 📝 Short Description
- 🔗 Direct link to the original article

---

### 🎨 Interactive Interface

- Dark Modern UI
- Sidebar Controls
- Card View
- Grid View
- Responsive Layout
- Loading Indicators
- Success Messages

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/A6dur/NewsReader.git
```

Move into the project

```bash
cd NewsReader
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```text
NEWS_API_KEY=YOUR_API_KEY
```

Run the application

```bash
streamlit run app.py
```
